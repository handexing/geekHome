# :dancers: 极客屋

## 项目介绍

极客屋是极客们的社区。在这里大家能够多多分享自己正在做的有趣事物、最近的学习、交流想法，在这里找到朋友认识更多有趣的人。

## 项目涉及
- springboot
- nginx

## 静态网页

[geekhome](https://github.com/handexing/geek_page),采用动静分离，需安装nginx，[nginx配置](https://github.com/handexing/geekHome/tree/master/config)
